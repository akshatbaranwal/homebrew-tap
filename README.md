# homebrew-tap

Personal Homebrew tap for [Akshat Baranwal](https://github.com/akshatbaranwal)'s macOS apps.

## Casks

| Name | Description |
|------|-------------|
| [simple-caffeine](https://github.com/akshatbaranwal/simple-caffeine) | Menu bar app that keeps your Mac awake |

## Usage

```sh
brew tap akshatbaranwal/tap
brew install --cask simple-caffeine
```
